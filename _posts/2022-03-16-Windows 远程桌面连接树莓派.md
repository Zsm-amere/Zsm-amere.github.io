# Windows 远程桌面连接树莓派

在 Raspibian 中安装`xrdp`服务，命令如下：

> ```shell
> sudo apt-get install xrdp
> ```

重启`xrdp`服务

> ```shell
> sudo /etc/init.d/xrdp restsrt
> ```
